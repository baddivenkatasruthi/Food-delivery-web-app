# 🍔 Food Delivery Web App  
## Simplified Campus Canteen Food Ordering System

## 📌 Project Overview
This project is a **Food Delivery Web Application** designed as a **simplified food ordering system for a campus canteen**.  
It enables users to view the menu, place orders, and track order status, while the **Admin manages users, orders, and reports**.

The application is developed as a **team-based academic project** using the **MERN stack architecture**.

---

## 🎯 Problem Statement
Traditional campus canteens rely on manual ordering systems which lead to:
- Long queues during peak hours  
- Order mismatches and delays  
- Difficulty in tracking sales and popular items  

This project aims to **digitize the ordering process** to improve efficiency and transparency.

---

## 📦 Project Scope
- Online menu display  
- Add to cart and checkout  
- Order history tracking  
- Admin order management and reporting  
- Database-driven scalable system  

> Note: Delivery is limited to **canteen order preparation and status updates**.

---

## 🧩 System Modules

### 👤 User Module
- User registration and login  
- Browse menu  
- Add items to cart  
- Place orders  
- View order history  

### 🧑‍🍳 Canteen Module
- Menu management  
- View incoming orders  
- Update order status (Preparing / Ready)  

### 🛠️ Admin Module (Main Focus)
- User management  
- Canteen/restaurant management  
- Order monitoring  
- Sales reports and earnings analysis  

### 🔔 Common Features
- Order status updates  
- Basic notifications  

---

## ⭐ Admin Module – Detailed Description
The **Admin Module** acts as the central control unit of the system.

### Admin Functions:
- View and manage registered users  
- Monitor all placed orders  
- Update order status during peak hours  
- Generate daily and monthly sales reports  
- Calculate total canteen earnings  
- Analyze most ordered menu items  

This module ensures **smooth operation, transparency, and control**.
